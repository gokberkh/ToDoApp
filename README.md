# Django Todo App

# Used

- Framework - Django
- Database - Postgresql ( ElephantSQL )
- Javascript Library - Jquery 
- Css  - Bootstrap


# Doing

- Login
- User-Todo-Completed in database
- Added Todo
- Update Todo
- Completed/ Not Completed Todo
- Delete Todo

## Running

```shell
    python manage.py runserver

```
